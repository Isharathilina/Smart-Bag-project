# Smart_Children_Bag
## Technologies
* Nodemcu
* Arduino
* MQTT
* mfrc522 rfid reader
* Neo-6m GPS Module
* SIM 900 GSM module

## Project Description
Smart Bag for track children real-time location and monitor their bag's inside items. </br>
Bag Item reading by RFID. Use GSM module for send and receive location data trough SMS
